Major Project - REAL TIME ARDUINO-BASED CARDIAC DISEASE DETECTION USING NEURAL NETWORK

In this project, we applied a variety of machine learning algorithms to predict the development of heart disease in real time. We used AD8232 sensor and Arduino to collect ECG data of the paitent. In order to find the model with the highest accuracy for processing ECG Data, the Kaggle dataset was obtained and trained using various methods. The most accurate model was chosen for diagnosing cardiac problems in real-time. After collecting and processing the data using neural netwoek the system predicts wheather the paitent is having a normal heart beat or an abnormal one.

Mini Project - MINI CNC PLOTTER USING DVD WRITERS

In this project we built a low cost mini CNC plotter using two stepper motors from two DVD writers which can be easily controlled using computer software. This setup of hardware with a combination of G-code gives better accuracy and reduces the work load. G code makes it easy to find the information of locations of all stepper motor moving, as the status of our moving motor are directly seen on computer hence we can start or stop the machine whenever we are needed.By using this we can make Difficult and Complex Design on paper. We can have some changes on it and make it available for commercial use to create tools for cutting, grinding of soft material etc.
